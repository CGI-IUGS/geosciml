Metadata vocabularies Notes        
Thematic accuracy terms Confidence that specified value is accurate. Examples: high, standard, low. See ISO 19115 DQ_ThematicAccuracy  
Value type semantics Union elements have attribute that specifies the kind of element implemented by any particular instance of the union element. In this case, populated with string, allowed values are :"value" and "range". This is a flag to soft type a term or numeric property value as either a single value or a range value.  
Metadata scope terms Controlled vocabulary Codes for metadata hierarchy level. Values: “attribute”, “collectionSession”, “dataset”, “feature”, “feildSession”, “model”, “nonGeographicDataset”, “series”, “service”, other values allowed.
