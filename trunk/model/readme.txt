This directory contains the top level GeoSciML UML package (GeoSciML4.0.xml) 
under which all other GeoSciML UML packages sit.  All of the component GeoSciML
UML packages are in directories under https://www.seegrid.csiro.au/subversion/GeoSciML/trunk/
(eg, https://www.seegrid.csiro.au/subversion/GeoSciML/trunk/GeologicAge) and are copied
to the "model" directory for ease of use in Enterprise Architect via svn-external.

Please follow the instructions at:
https://www.seegrid.csiro.au/wiki/CGIModel/ConfiguringEnterpriseArchitectForGeoSciML 
to configure Enterprise Architect for HollowWorld and GeoSciML.
