This directory contains software-specific template models, that provide a convenient starting point for loading the current XMI version of documents representing the various packages of the GeoSciML UML model. 

For instructions for how to load these into Enterprise Architect, see https://www.seegrid.csiro.au/twiki/bin/view/CGIModel/ConfiguringEnterpriseArchitectForGeoSciML
