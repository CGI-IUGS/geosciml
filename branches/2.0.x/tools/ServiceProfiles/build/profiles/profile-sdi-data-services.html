This profile controls the discoverability of data access services according to common concepts of data custodianship withing a general Spatial Data infrastructure. To aid discovery of services using OGC Catalogue Services, a service must also have a corresponding metadata record that describes the service according to ISO 19115 element content and encoded in XML to the ISO 19139 Implementation Standard.
