GeoSciML.PhysicalProperties.PhysicalDescription
Subclasses:
Author:CGI Interoperability Working Group
Version:3.2.0
Notes:A class to describe the numeric physical properties of a geologic unit, earth material, or geologic structure. (eg; density, porosity, magnetic susceptibility, remanent magnetism). These properties are modelled here as scalar numeric values (swe:Quantity).

Vector and tensor physical properties are considered to be more applicable to located observations and should be delivered as SamplingFeature/Observations with associated geologic unit or geologic structure features.
