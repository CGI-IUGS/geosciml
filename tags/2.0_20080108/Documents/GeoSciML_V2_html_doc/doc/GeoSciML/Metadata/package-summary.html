Package GeoSciML/Metadata

The GeoSciML Metadata package holds a single diagram that shows the metadata links from various GeoSciML classes. For metadata support, GeoSciML refers to the (externally maintained) ISO 19115 metadata package. The GeoSciML metadata package corresponds to the XML schema document that defines the necessary association type. Pending GeoSciML migrating to GML v3.2, the XML Schema also contains a stub schema representing the actual metadata elements.
