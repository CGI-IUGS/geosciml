A GeoServer configuration for GroundWaterML 2 that uses GeoSciML 4 basic (gwmlb) for a Geologic unit feature (GW_Aquifer) and GeoSciML 4 extension (gwmlx) for the gsmlb:gbUnitDescription/gsmlx:GeologicUnitDescription/gsmlx:unitThickness property. 
The associated files/folders for included (FEATURELINK) files and namespaces are excluded.
